📖 安装指南

🪟 Windows 安装步骤

1 下载安装包 — 点击上方"下载 .exe 安装包"按钮，将安装程序保存到本地。
2 运行安装程序 — 双击下载的 .exe 文件，在安装向导中选择你想要的安装目录，勾选"创建桌面快捷方式"。
3 开始使用 — 安装完成后桌面会出现 ImageCompress 图标，双击启动即可离线使用全部功能。

📱 Android 安装步骤

1 下载 APK — 点击上方"下载 APK 安装包"按钮，或在手机浏览器中打开本页面下载。
2 允许安装 — 下载完成后，系统会提示"允许来自此来源的安装"，点击"设置"→ 开启"允许安装未知应用"。
3 打开应用 — 安装完成后，在桌面或应用抽屉中找到 ImageCompress 图标，点击即可使用。

常见问题

桌面版和网页版功能一样吗？

功能完全一致，都包含图片压缩、去水印、证件照换底色三大核心功能，且均为本地处理不经过服务器。桌面版额外优势：无需打开浏览器、文件系统直接交互、窗口始终置顶。

安装包安全吗？会不会有病毒？

安装包由我们官方构建和签名，不含任何恶意代码。所有图片处理在你自己的设备上完成，不会上传任何数据到服务器。如果你有顾虑，可以继续使用网页版，效果完全相同。

Windows 安装时可以自己选安装位置吗？

可以。安装向导中会显示安装目录选择页面，默认路径为 C:\Program Files\ImageCompress，你可以点击"浏览"选择其他目录。

后续如何更新到新版本？

应用内置了自动更新检测功能，有新版本时会弹出通知。你也可以随时访问本下载页面获取最新安装包，覆盖安装即可（不会丢失设置）。

有 macOS 或 iOS 版本吗？

macOS 和 iOS 版本目前正在开发中，敬请期待。即将推出。同时你也可以直接使用 网页版，在 Safari 或 Chrome 中打开即可使用全部功能。
