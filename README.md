# ImageCompress - 在线图片压缩工具

[![Website](https://img.shields.io/badge/官网-www.zaixianyasuo.cn-blue)](https://www.zaixianyasuo.cn)

免费在线图像压缩工具，无需上传服务器，本地处理，保护隐私。

## 功能

- **图片压缩** — 支持 PNG、JPG、WebP 格式批量压缩，画质损失小
- **证件照换底色** — 一键更换证件照背景颜色
- **去水印** — AI 智能去除图片水印

## 技术栈

- 前端：原生 HTML/CSS/JS，无框架依赖
- 后端：Python FastAPI + MySQL
- 图像处理：Pillow, OpenCV
- 部署：阿里云 ECS + Nginx + Cloudflare CDN
